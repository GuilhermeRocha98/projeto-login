# Projeto Login

Este é um projeto simples de login desenvolvido em Java com as seguintes tecnologias:

- Java
- Spring Framework
- Spring Boot 3.0.2
- Spring Security
- Spring Data JPA
- PostgreSQL
- HTML
- CSS
- JavaScript

## Funcionalidades

- Autenticação de usuários(É necessário registrar o usuário diretamente no banco de dados)

## Pré-requisitos

- Java 8 ou superior
- Maven
- PostgreSQL
